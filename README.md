# Copia-Instagram
- Nova versão (completo)

Eu desenvolvi esse projeto enquanto estava no curso de desevolvimento web na Prepara curso.

Deste projeto eu aprimorei:

- Tags HTML
  - Ancora
     - Utilizada principalmente com Href para uma ligação de uma pagina para outra.
  - Div
     - Usado para dividir seções.
  - H1 até H6
     - São tags de textos.
  - Image
     - Utilizado para carregar imagens
  - Input
     - Utilizado para criar controles interativos.

- Atributos
  - Src
     - Encaminha algo externo como arquivo.
  - Href
     - link capaz de viajar de paginas a outra.
  - Id
     - Utilizado para indentificar um unico elemento.

- Display
  - Flex
     - Capaz de alinhar e organizar os itens dentro.

- CSS
  - Style
     - Utilizado para modificar o Corpo(body) ou qualquer seção
